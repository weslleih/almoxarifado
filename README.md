Almoxarifado
============

Simples controle de estoque.

Recursos
------------
- Multi usuário com níveis hierárquicos 
- Grupos de produtos
- Fornecedores com dados de contato
- Consumidores
- Relatórios configuráveis
- Log de alterações

Requisitos
------------
### Servidor
- Apache 2.2
- PHP 5.3
- MySQL 6

### Cliente
- Internet Explorer 10
- Mozilla Firefox 4
- Google Chrome 22
